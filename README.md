# Damas Food Specialties - Restaurant Management System

موقع متخصص لإدارة طلبات المطعم مع واجهة للزبائن ولوحة تحكم للإدارة.

## المميزات الرئيسية

✅ **واجهة زبون احترافية** - تصفح المنيو والطلب بسهولة
✅ **لوحة تحكم إدارية** - إدارة الطلبات والمستخدمين
✅ **قاعدة بيانات دائمة** - PostgreSQL لحفظ البيانات بشكل آمن
✅ **دعم اللغات** - العربية والإنجليزية
✅ **نظام مصادقة** - تسجيل دخول آمن للمسؤولين

## المتطلبات

- Node.js v18 أو أعلى
- npm أو pnpm
- PostgreSQL

## التثبيت والتشغيل

### 1. استنساخ المستودع
```bash
git clone <repository-url>
cd File-Reader-1
```

### 2. تثبيت التبعيات
```bash
npm install
```

### 3. إعداد قاعدة البيانات
```bash
# إنشاء قاعدة البيانات
npx drizzle-kit push:pg
```

### 4. تشغيل الخادم
```bash
npm run dev
```

سيبدأ الخادم على `http://localhost:3000`

## البنية الأساسية

```
├── client/              # الواجهة الأمامية (React + Vite)
│   ├── src/
│   │   ├── pages/      # صفحات التطبيق
│   │   ├── components/ # المكونات
│   │   ├── store/      # إدارة الحالة
│   │   └── main.tsx    # نقطة الدخول
│   └── index.html
├── server/             # الخادم الخلفي (Express + TypeScript)
│   ├── index.ts        # ملف الخادم الرئيسي
│   ├── routes.ts       # المسارات
│   ├── auth.ts         # المصادقة
│   ├── storage.ts      # إدارة البيانات
│   └── db.ts           # إعدادات قاعدة البيانات
├── shared/             # ملفات مشتركة بين الخادم والعميل
└── package.json
```
